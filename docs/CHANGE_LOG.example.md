# CHANGE_LOG.example.md

## 用途

CHANGE_LOG.md用于记录重要 AI 代码修改，方便回顾、审计和回滚。
每次修改后建议按下面记录模板的格式追加到CHANGE_LOG.md中。

## 记录模板

```md
### YYYY-MM-DD - 修改标题

#### 修改目标

说明本次要解决的问题。

#### 修改文件

- `path/to/file.py`：修改原因
- `path/to/test.py`：验证原因

#### 未修改范围

说明哪些关键区域没有动。

#### 验证方式

说明运行了哪些测试，或如何手动验证。

#### 剩余风险

说明仍然存在的不确定点。
```
